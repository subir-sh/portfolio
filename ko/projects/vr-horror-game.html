폐병원 탈출

Meta Quest 2용 Unity 기반 VR 탈출 호러 게임으로, 몰입형 연출과 짧은 플레이 경험에 집중했습니다.

프로젝트 개요

폐병원 탈출은 Meta Quest 2를 대상으로 Unity에서 개발한 VR 탈출 호러 게임입니다. 폐병원을 배경으로 긴장감, 몰입감, 탈출 중심 진행 구조에 초점을 맞춘 짧은 1인칭 호러 경험을 설계했습니다.

대학 수업 프로젝트로 개발했으며 약 10분 플레이 길이를 목표로 했고, 최종 쇼케이스 현장 학생 투표에서 1위를 기록했습니다. 저는 UI, 시각 효과, 카메라 전환, 컷신 흐름 등 연출 관련 파트를 주로 담당했습니다.

담당 내용

  • 몰입형 시각 효과, 사운드 연동 연출 요소, UI/UX 컴포넌트를 구현했습니다.
  • 카메라 전환과 후처리 기반 씬 연출을 구현했습니다.
  • 오프닝/엔딩 컷신과 게임 오버/클리어 화면을 제작했습니다.
  • GitHub 기반 팀 워크플로우로 협업했습니다.

설계상 과제

핵심 과제는 Unity 에디터 화면이 아니라 실제 헤드셋 환경에서 연출 요소가 자연스럽게 보이도록 맞추는 것이었습니다. 로컬 Game View에서 괜찮아 보이던 UI 위치·크기·씬 전환이 실제 VR에서는 다르게 느껴지는 경우가 많았습니다.

반복적인 실제 기기 테스트를 통해 캔버스 설정, 트랜스폼, 연출 흐름을 조정해 헤드셋에서도 인터페이스와 컷신이 안정적으로 동작하도록 개선했습니다. 이 프로젝트를 통해 VR 연출은 일반 화면 기반 씬처럼 다루지 말고, 플레이어의 기기 내 시점에서 검증해야 한다는 점을 배웠습니다.

게임플레이 미리보기

어두운 분위기 연출과 몰입 중심 경험 설계를 보여주는 VR 호러 환경의 실제 게임플레이 장면입니다.

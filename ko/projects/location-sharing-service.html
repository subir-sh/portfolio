오고있니

도착 지점만이 아니라 만나러 가는 과정을 기록하는 세션 기반 실시간 위치 공유 서비스입니다.

프로젝트 개요

오고있니는 커플이 서로 만나러 가는 동안 위치와 이동 경로를 실시간으로 공유하는 모바일 웹 서비스입니다. 단순 위치 추적보다, 만나러 가는 과정 자체를 함께 볼 수 있는 경험에 초점을 맞췄습니다.

와플스튜디오 주최 해커톤 2026에서 개발해 3위를 수상했습니다. 저는 백엔드 세션 모델링과 실시간 통신 흐름을 맡아, 위치 업데이트, 세션 기록, 인증 기반 접근 제어가 안정적으로 동작하도록 구현했습니다.

담당 내용

  • 세션과 위치 포인트 이력을 저장하는 DB 스키마를 설계했습니다.
  • 실시간 세션 업데이트를 위한 백엔드/프론트엔드 WebSocket 흐름을 구현했습니다.
  • WebSocket 핸드셰이크에 JWT 인증을 연결하고, 세션 참여자 검증을 추가했습니다.
  • HTTP로 기존 이력을 불러오고 WebSocket으로 새 좌표를 받아, 새로고침 후에도 지도 상태가 복원되도록 구현했습니다.
  • 세션 생성/수락 시 소켓이 자동으로 시작되고, 세션 종료까지 이어지는 흐름을 구현했습니다.

설계상 과제

핵심은 WebSocket 연결 자체가 아니라, 세션의 시작부터 종료까지 이어지는 흐름을 안정적으로 설계하는 것이었습니다. 이 서비스에서 WebSocket은 인증, 세션 입장, 이력 로딩, 실시간 좌표 업데이트, 세션 종료와 함께 동작해야 했습니다.

기존 위치 이력은 HTTP로 불러오고, 이후 좌표 업데이트는 WebSocket으로 받도록 나눴습니다. 덕분에 새로고침이나 재연결 상황에서도 같은 세션 상태를 이어갈 수 있었습니다.

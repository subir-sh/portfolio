XR 학부 연구 인턴

객체 기반 XR 오버레이, 공간 정렬, 출력 품질 평가를 다룬 학부 연구 인턴십입니다.

학부 연구 인턴 / HCS Lab

역할 / 2D-3D 그라운딩, XR 오버레이 구현, 생성 품질 평가

기술 / Unity, OpenXR, Gemini

안내

논문 투고 후 익명성 고려가 필요해, 프로젝트의 상세 정보는 의도적으로 제외했습니다.

담당 내용

  • 이미지 기반 지시 대상 탐지 결과를 객체 연결 XR 오버레이로 이어주는 2D-3D 그라운딩 작업을 수행했습니다.
  • 탐지된 지시 대상을 월드 앵커 기반 시각 타깃에 매핑하는 XR 오버레이 컴포넌트를 구현했습니다.
  • 객체 기반 시각 제시를 위해 이미지 공간 탐지, 카메라 투영, 월드 공간 배치 사이의 정렬을 처리했습니다.
  • AI가 생성한 멀티모달 출력이 유효한지, 장면 맥락과 의도한 응답에 맞는지 평가했습니다.

설계상 과제

핵심은 1인칭 시점 이미지에서 검출한 2D 지시 대상을 실제 물체 위에 안정적으로 보이는 XR 오버레이로 바꾸는 일이었습니다. 시각 단서는 단순한 화면 주석이 아니라 실제 물체 위치와 맞아야 했고, 사용자 움직임과 모델 출력 타이밍에도 맞춰져야 했습니다.

이를 위해 이미지 탐지 좌표, 카메라 투영, 월드 앵커 좌표를 연결해야 했습니다. 저는 장면 맥락에 맞는 시각 제시와 평가가 필요한 구간에서 그라운딩/오버레이 파이프라인을 맡았습니다.
